# 🕉️ GitaGuide AI

## 🌟 Overview

GitaGuide AI is an AI-powered Bhagavad Gita mentor that provides personalized spiritual guidance using mood detection and Gemini AI.

---

## ✨ Features

- 🤖 AI Guidance using Gemini
- 📖 Bhagavad Gita Verse Recommendation
- 🧠 Mood Detection
- 🎯 Action Suggestions
- 💭 Reflection Questions
- 💬 Interactive Chat Interface

---

## ⚙️ Tech Stack

- Python
- Streamlit
- Gemini AI
- JSON Knowledge Base

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
